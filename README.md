# skynet_wsdemo
skyent websocket 测试

# 概要
    1. main.lua 程序入口
    2. ws_server websocket服务
    3. ws_client 模拟客户端
    
# 启动
    1. 单节点启动
        cd skynet_wsdemo
        bin/skynet conf/config
    2. 独立启动
        服务端启动:
            cd skynet_wsdemo
            bin/skynet conf/config_ws
        客户端启动
            cd skynet_wsdemo
            bin/skynet conf/config_client
